## Làm sao để chạy

**Cài đặt module**<br>
Cài đặt các module cần thiết

```command
npm i
```

**Run dev**<br>
chạy ở chế độ dev tự động reload kho có thay đổi file

```command
npm run dev
```

**Other command**

-   Build

```command
npm run build
```

-   Start (Yêu cầu có bản build trước)

```command
npm start
```
