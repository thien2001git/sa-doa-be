## Làm sao để chạy

**Get submodules**<br>

```bash
git submodule update --init --recursive
```

**Cài đặt module**<br>
Cài đặt các module cần thiết

```bash
npm i
```

**Run dev**<br>
chạy ở chế độ dev tự động reload kho có thay đổi file

```bash
npm run dev
```

**Other command**

-   Build

```bash
npm run build
```

-   Start (Yêu cầu có bản build trước)

```bash
npm start
```
